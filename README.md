# ReportsGenerator

This project is an sample of report generator from .csv file.

## Configuration

- Run app
  
  ```sh
  iex -S mix
  ```

- Run tests

  ```sh
  mix test
  ```

- Update deps

  ```sh
  mix deps.get
  ```
